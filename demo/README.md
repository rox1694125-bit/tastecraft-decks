# Demo

The demo directory is for public, synthetic walkthroughs of TasteCraft Decks. It should help contributors understand the intended workflow without exposing real customer material or adding binary assets.

## Rules

- Use fictional organizations only.
- Keep examples text-only unless a future release policy explicitly approves reviewed assets.
- Do not add real logos, screenshots, private decks, personal data, or proprietary brand systems.
- Do not place real input JSON under `demo/sample-inputs/`.
- Label synthetic examples clearly.

## Current Contents

- `gallery/`: markdown gallery notes for fictional deck scenarios.
- `sample-inputs/`: synthetic JSON fixtures that validate against the V1 schemas.
- `sample-outputs/`: reserved for generated outputs after release review.
- `screenshots/`: reserved for reviewed screenshots if a future release allows them.

Start with [demo/gallery/README.md](gallery/README.md).
