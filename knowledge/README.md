# Knowledge

The knowledge directory stores public, reusable guidance for TasteCraft Decks. It is not a private research archive and must not include customer material.

## Contents

- `design-patterns/`: reusable presentation patterns and layout reasoning.
- `anti-patterns/`: common deck failures and how to avoid them.
- `examples/`: fictional briefs and synthetic source material.
- `brand-kits/`: public synthetic brand-kit examples only.
- `research/`: public research notes if future contributors add them with proper attribution.

## Public Safety Rule

Everything in this directory should be safe to publish, fork, package, and redistribute. Do not add real logos, private decks, customer data, unreleased screenshots, or files from `knowledge/brand-kits/_local/`.
