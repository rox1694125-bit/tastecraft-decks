# Maintainers

This project uses maintainer-led review. The initial maintainer group is represented by the `TasteCraft Decks contributors` copyright holder in the MIT license until named maintainers are added.

## Maintainer Duties

- Review pull requests.
- Keep release notes accurate.
- Enforce data and brand policy.
- Request ADRs for lasting decisions.
- Tag releases only after the release checklist is complete.

## Adding Maintainers

To add a maintainer:

1. Open a governance issue.
2. Summarize the contributor's project history.
3. Confirm they understand the synthetic demo and data policy.
4. Record the decision in this file.
