# Synthetic Neutral Brand Kit

This is a fictional public brand-kit example for TasteCraft Decks demos. It does not represent a real organization and must not be used to imitate a real logo or brand system.

## Identity

- Name: Synthetic Neutral Kit.
- Use case: public demo decks that need a restrained, professional look.
- Brand mode: white-label or no-logo.
- Logo policy: text-only name on cover if needed; no symbol mark.

## Palette

| Token | Hex | Purpose |
| --- | --- | --- |
| Background | `#F7F7F2` | Slide background. |
| Surface | `#FFFFFF` | Tables and chart panels. |
| Text | `#1F2933` | Main text. |
| Muted text | `#5E6A75` | Captions and secondary labels. |
| Primary | `#256D85` | Structural emphasis. |
| Secondary | `#6A8D73` | Supporting categories. |
| Accent | `#D29A3A` | Warnings and decision highlights. |

## Typography Direction

- Heading style: crisp sans serif, medium weight.
- Body style: readable sans serif, regular weight.
- Density fit: balanced or reading-first.

## Imagery Direction

- Prefer diagrams, charts, and abstract shapes.
- Avoid real photography.
- Avoid real logos, product packages, landmarks, or people.

## Forbidden Elements

- Real company marks.
- Real brand color systems.
- Customer names.
- Product screenshots.
- Generated mascot or logo marks that resemble existing brands.
