# Deck Anti-Patterns

Use this file as a review aid when improving TasteCraft skill instructions and demo examples.

## Template Theater

Symptoms:

- Looks polished but every slide says the same thing.
- Decorative cards overpower the actual decision.
- Layout style is chosen before audience and goal.

Correction:

- Write page objectives first.
- Remove decorative elements that do not support the argument.
- Tie visual hierarchy to the decision path.

## Flattened PPTX

Symptoms:

- Every slide is exported as one image.
- Text and charts cannot be edited.
- File looks correct but is hard to maintain.

Correction:

- Keep PPTX objects editable.
- Use images only for intentional visual assets.
- Document any unavoidable rasterization.

## Synthetic Example Drift

Symptoms:

- Fictional examples start resembling real companies.
- Metrics or names are copied from public case studies.
- Prompts ask for a recognizable brand style.

Correction:

- Rename entities to obviously fictional names.
- Invent metrics from scratch.
- Add forbidden elements to prompt packs.

## Contrast Debt

Symptoms:

- Elegant palette but low readability.
- Muted labels disappear on projectors.
- Accent colors carry meaning without labels.

Correction:

- Check contrast early.
- Use text labels with color-coded states.
- Prefer readable hierarchy over palette novelty.

## Overfilled Executive Pages

Symptoms:

- Board or investor slides contain too many parallel claims.
- Charts lack a decision headline.
- The audience cannot tell what action is requested.

Correction:

- Put the decision in the title or subtitle.
- Move supporting material to appendix-heavy pages.
- Use one primary chart or argument per slide.
